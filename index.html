Portfolio

"Lyft"


Built using MongoDB, Node.js, and Lyft and Google Map's API, this app allows users to drag pins to indicate their pick up location and destination. All ride types (lyft, lyft-line, lyft-plus) prices populate taking into account surcharge pricing.

"Travel Log"


Built using Node.js and Google Map's API, Travel Log allows users to keep track of all the destinations they've traveled to as well as a bucketlist of where they'd like to go.
